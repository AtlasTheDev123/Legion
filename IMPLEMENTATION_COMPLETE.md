## 🚀 Quick Start Guide

### Run Main Demo
```bash
python run_nexus_ai.py
```

### Run Integration Examples
```bash
python examples/ai_integration_examples.py
```

### Use in Python Code
```python
from agents.orchestrator import NexusAIOrchestrator

# Initialize
ai = NexusAIOrchestrator()

# Execute task
result = ai.execute_task("Optimize database queries")
print(f"Confidence: {result['confidence']:.2%}")

# Analyze code
analysis = ai.analyze_code(code_string, "python")
print(f"Complexity: {analysis['code_analysis']['complexity_score']}")

# Solve problem
solution = ai.solve_problem("How to scale to 1M users?")
print(f"Strategy: {solution['approach']}")

# Learn from experience
ai.core_ai.learn(
    "Implemented caching",
    "Performance improved 80%",
    importance=0.9,
    tags=["optimization"]
)

# Optimize system
report = ai.optimize_system()
```

### Start API Server
```bash
cd api
uvicorn nexus_api:app --reload --host 0.0.0.0 --port 8000
```

Then access:
- **API Docs**: http://localhost:8000/docs
- **API**: http://localhost:8000/api/
- **WebSocket**: ws://localhost:8000/ws
