# Onboarding Guide (dev)

- Install Python 3.10+
- Create a venv: `python -m venv .venv` and `.
venv\Scripts\Activate.ps1`
- Install deps: `pip install -r requirements.txt`
- Run API: `uvicorn dashboard.backend.main:app --reload`
- Use `python bot/vex_x_bot.py <function> --params '{}` to simulate calls
